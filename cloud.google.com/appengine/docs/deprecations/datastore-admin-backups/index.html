Datastore Admin Backups Shutdown

The Datastore Admin backup feature was deprecated on February 28, 2018, and will be shut down entirely on February 28, 2019. It is replaced by the managed export and import service for Cloud Datastore.

Migrating to the managed export and import service

To begin using the managed export and import service, see Exporting and Importing Entities. This page also points out differences between Datastore Admin backups and managed exports.

Shutdown timetable

With any deprecation there is a follow-up decommission where the API or service is formally removed. The following table lists the shutdown schedule for the Datastore Admin backup feature:

Date Event
February 28, 2019 The Datastore Admin backup feature will no longer be available.
