Local data around the world

Get data from the same database used by Google Maps. Places features over 100 million businesses and points of interest that are updated frequently through owner-verified listings and user-moderated contributions.

Place picker

Add the built-in place picker UI widget to your app, so users can choose from a set of nearby places displayed on a map.

Place autocomplete

Automatically fills in the name and / or address of a place as you type.

Place details

Retrieve rich details about a place, including name, address, phone number, website link and more.
