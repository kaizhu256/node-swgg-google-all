Introduction

The Ad Experience Report API lets developers build applications that query the Google Ad Experience Report. The API can be used to get a list of sites that have ad experiences that violate the Better Ads Standards, or to get a summary of the ad experience rating of a site. The response also contains a link to the full Ad Experience Report for publishers to get more details.

Note: If a site has not been reviewed, or if it has been reviewed but its status is “Passing,” then it’s not listed in the Ad Experience Report API.

For more information about a site’s review status and ad experiences, see the Ad Experience Report. For more information about the Report, visit the Web Tools Help Center.
